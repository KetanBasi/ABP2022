Fungsi Console Lainnya

  • console.clear()

    Clear console

  • console.count(label)

    Count berapa kali pemanggilan count() dilakukan.

    Jika label diisi string, count() akan menghitung berapa kali fungsi ini dipanggil dengan label tersebut.

  • console.countReset(label)

    Reset counter dari console.count(label)

  • console.debug()

    Output message to web console at debug log level

  • console.error()

    Output message to web console at error log level

  • console.group()

    Creates new inline group inside the web console log, pesan setelahnya akan tampil diindentasi satu level hingga console.groupEnd() dipanggil

  • console.groupCollapsed()

    Grouping seperti console.group(), namun group ditampilkan collapsed

  • console.groupEnd()

    Mengakhiri grouping / indenting dari console.group()

  • console.info()

    Output message to web console at info log level

  • console.log()

    Output message to web console

  • console.table()

    Menampilkan data tabular dalam tabel

  • console.time(nama)

    Starts a timer, bisa diberi nama. Ketika console.timeEnd() dengan nama yang sama, waktu akan ditampilkan.

  • console.timeEnd(nama)

    Stops console.time(nama)

  • console.timeLog()

    Menampilkan current value dari console.time() yang baru dimulai

  • console.warn()

    Output a warning message to web console

5 method manipulasi array

Array: var fruits = ["apple", "banana", "cherry"];

  • As a string toString()

    Convert array to string, separated by comma

    console.log(fruits.toString); // apple,banana,cherry

  • Slicing slice()

    console.log(fruits.slice(0,2)); // ["apple", "banana"]

  • Filter filter()

    console.log(fruits.filter((fruit) => fruit.charAt(0) === 'a')); // Array ["apple"]

  • Merging / concat concat()

    console.log(fruits.concat(["guava", "lemon"])); // Array ["apple", "banana", "cherry", "guava", "lemon"]

  • Reverse reverse()

    fruits.reverse();

    console.log(fruits); // Array ["cherry", "banana", "apple"]



Eksperimen method manipulasi array pada object

Object: var apple = { name: "apple", color: "red" };

  • toString()

    apple.toString(); // "[object Object]"

  • slice()

    apple.slice(...); // (error not a function)

  • filter()

    apple.filter(...); // (error not a function)

  • concat()

    apple.concat(...); // (error not a function)

  • reverse()

    apple.reverse(); // (error not a function)
